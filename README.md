# Status-board
군복무 시절 사용했던 현황판의 재현 및 개선 버전
## 기능
#### 열외 사유 버튼과 병사 버튼을 통해 색깔로 직관적이게 열외 사유를 파악할 수 있게 하였습니다.
#### 입대일자를 기반으로, 자동으로 공군 기준 진급일과 전역일을 계산하여 전역자는 목록에서 제외하고, 계급을 자동으로 표시하게 하였습니다.
#### 간부 목록이 추가되었습니다. 간부는 별도로 계급을 변경할 수 있습니다.

## 추가할 점
#### 병사도 진급누락 등을 고려해 계급을 변경할 수 있는 기능을 추가할 것입니다.
#### 프로그램을 껐다가 켜도 유지되도록 브라우저 localstorage를 이용할 것입니다.
#### 지금은 병사가 추가된 일자를 기반으로 한 ID 기준으로 정렬되어 표시되나, 추후 입대일 기준으로 표시하고자 합니다.
